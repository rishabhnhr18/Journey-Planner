# DelivIQ Analytics Dashboard

A production‑ready delivery & inventory intelligence dashboard built with **Next.js 16**, **React 19**, **TypeScript**, and **Recharts**.  
Designed for logistics operations to monitor inventory, customer behaviour, territory performance, salesperson efficiency, and AI‑powered forecasting.

## ✨ Features

- **Inventory Management** – stock levels, turnover, low stock alerts.
- **Customer Intelligence** – top customers, churn risk, order value trends.
- **Territory Analytics** – performance by zone, success rates, radar charts.
- **Salesperson Leaderboard** – deliveries, success rate, revenue, route efficiency.
- **AI Planning** – inventory & order forecasts, monthly delivery plans, route optimisation.
- **Advanced AI (Research)** – dynamic territory design, smart routing, in‑visit recommendations.
- **User Profile** – view/edit account details.

## 🚀 Tech Stack

- [Next.js 16](https://nextjs.org/) – React framework (App Router)
- [React 19](https://react.dev/) – UI library
- [TypeScript](https://www.typescriptlang.org/) – type safety
- [Recharts](https://recharts.org/) – charting library
- [Lucide React](https://lucide.dev/) – icons
- [Tailwind CSS](https://tailwindcss.com/) – styling (via CSS variables)
